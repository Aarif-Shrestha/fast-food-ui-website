#  Fast Food Website

This is a responsive frontend website for a fast food restaurant built using HTML and CSS.

##  Project Overview

The website includes a simple and clean design for a fast food restaurant, showcasing menu items, a homepage banner, and basic navigation.

##  Features

- Homepage with hero section
- Food menu/items section
- About section
- Contact section
- Clean UI layout

##  Tech Stack

- HTML
- CSS

##  What I Learned

- Structuring a multi-section webpage
- Styling layouts using CSS
- Designing a basic restaurant UI

##  Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Aarif-Shrestha/fast-food-ui-website.git
```

2. Open the project folder.
3. Open `index.html` in your browser.
